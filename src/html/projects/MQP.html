Swarm Robotics MQP

Swarm construction researchers have generally developed solutions to build structures in lab environments, not for marketable industry applications. The intent of this project is to design a system to traverse and install shingles on a flat roof. This new application requires roboticists to research into the following problems:

  1. Algorithm design to efficiently shingle a roof that scales well as the number of robots on the roof increases.
  2. Robot localization on a roof with limited pre-installed landmarks and sensors.
This project will extend previous work done on a construction platform that built 3-D structures out of “smart” cubes using bipedal “inchworm” robots. The smart cubes will be replaced with shingles that can store data to communicate with robots in the swarm. These inchworms are adapted to manipulate and traverse across shingles instead of 3-D blocks. The data storage capabilities of the shingles allow the robots to store information in the environment to localize, and assign tasks without the need for a centralized system.
