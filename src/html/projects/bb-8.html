Operational BB-8

The aim of this project is to create a replica of BB-8 from the Star Wars universe. The main struggle with this project is in creating the internal drive system for robot. All external 3d printed parts were collected from the BB-8 builders club forum along with the inner sphere by Angelo Cavallo, the files for which can be found here. The robot utilizes a holonomic drive train to move the main ball around. The drive system is currently being manufacured and tested. The armature for controling the head is still in development, but will hold the head to the main ball with magnets.
